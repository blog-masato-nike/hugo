Web会議システム構築（第2回 Linux（Ubuntu）へのJitsi Meetのインストール）

2020-05-01 Masato Nike

Linux(Ubuntu)にWeb会議システム『Jitsi Meet』をインストールします。 前提として、以下を実施しているものとします。 第1回 Linux（Ubuntu）サーバの構築

1. SSHの設定

(1) アプリケーションの一覧を表示してください。 (2) [端末]をクリックさせてください。 以下、端末にコマンドを入力することになります。 (3) SSHを利用できるようにします。以下のコマンドを入力してください。

sudo apt update
sudo apt upgrade
sudo apt install openssh-server
sudo /etc/init.d/ssh restart
sudo ufw status
sudo ufw allow proto tcp from 192.168.100.0/24 to any port 22
sudo ufw enable

(4) rootでのログインを拒否するように設定します。 【コマンド】

sudo vi /etc/ssh/sshd_config

【編集内容】

PermitRootLogin no
